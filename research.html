Research work and Projects

My Research interests primarily lie in Deep Learning based approaches to Computer Vision. The current problem which I am very actively pursuing is finding ways to extract accurate contextual information from a scene. Solutions to this problem can find application in a lot of fields with robotics and autonomous cars being the primary ones in my opinion. The following are a few projects that I've built over the course of my undergraduate education.

Video upscaling application using Generative Deep learning
models like SRGAN's (Ongoing)

When I read about Nvidia's DLSS 2.0 technology for gaming, I was very impressed. I liked the idea of downscaling things and then dynamically upscaling it and thus saving bandwidth. This sparked in me an idea to develop an application that can downscale videos on one end and send it across the network with low bandwidth requirements, and then upscale it on the other end. I started building such a system using SRGANs. The code for my prototype is available down below. One prominent application for this system would be to bring online educational content to rural areas with poor connectivity. If you can think of a better way of building such a system, feel free to reach out to me via my email below.
PS: The funny rotation of my head is to see if the GAN can properly upscale my face from different angles. Looks like it's been doing a fair job :)

Jatayu: An AI based Crime Tagging system

Motivated by a desire to build something that can make lives safer for everyone, I decided to work on this project with my team. With Jatayu, we repurposed the CCTV cameras from mere video recording machines into something that can detect an ongoing violent event. Using a network of such cameras, we can also track absconded criminals using the face recognition feature of Jatayu. We finished in top positions in SIH 2020 and IEEE Makerfair 2020 with this project. I was reponsible for the Deep Learning modules of the project.

Amrita CORE Lab Internship (May 2020 - Aug 2020)

The problem at hand was to map the learning curve of students in computer programming based on the kind of errors(Syntax,Semantic,Runtime,Logical) they made in their programs. Amrita's own coding platform HPOJ was used to collect the data (100,000+ samples). I preprocessed the data, extracted the information about the kind of errors a particular student made in a particular program and thus brought structure to the data. I also implemented a graph isomorphism based code similarity detection program to assign labels (student_id) to the unlabelled programs.

A simple Music Genre Recommender System

This Music Genre recommender system didn't exactly use any recommendation system-based algorithms like collaborative filtering etc. It was built using the Spotify dataset on kaggle and uses the concepts of Principle Component Analysis and Cosine Distance to recommend new genres based on the similarities between genres.
